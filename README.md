# Go-JS-Utils

## Desc

- Using js method in Golang

## Install

```
    go get -u github.com/zkfmapf123/go-js-utils
```

## Object Method

```sh
    ./object.go
```

|golang|js-method|
|---|---|
|OKeys|Object.keys|
|OValues|Object.values|
|OEntries|Object.entries|
